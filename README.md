# 🔐 Admin Panel — Система управления доступом

Современная и производительная панель администратора для управления пользователями, контроля проходов и мониторинга системы безопасности. Проект разработан с акцентом на **UX/UI**, производительность и отличную работу на любых устройствах.

## ✨ Основные возможности

- 📊 **Dashboard:** Наглядная статистика по активным пользователям, блокировкам и проходам за текущий день.
- 👥 **Управление пользователями:** Создание, удаление, а также "заморозка" (блокировка) учетных записей в реальном времени.
- 🛤 **Мониторинг проходов:** Детальный лог всех событий с фильтрацией по сканерам, результатам и ФИО.
- 🛡 **Журнал безопасности:** Полный аудит действий администраторов и системных событий.
- 🎨 **Брендинг:** Оформление в современном корпоративном стиле.
- 🌐 **Локализация:** Полная поддержка русского и английского языков (**i18next**).
- 🌓 **Темная тема:** Поддержка светлого и темного режимов с сохранением в `LocalStorage`.
- 📱 **Полный адаптив:** Интерфейс полностью оптимизирован под мобильные устройства, планшеты и десктопы.

---

## 🛠 Технологический стек

- **React 18** — мощная основа интерфейса.
- **Vite** — молниеносная сборка и разработка.
- **TypeScript** — надежность и строгая типизация.
- **Redux Toolkit & RTK Query** — современное управление состоянием и работа с API.
- **Vanilla CSS (Modules)** — чистая, производительная и модульная стилизация.
- **FSD (Feature-Sliced Design)** — продуманная архитектура для масштабируемости.

## 📱 Адаптивный дизайн

Мы уделили особое внимание тому, чтобы панель администратора была удобной везде:

## 🚀 Быстрый старт (Локально)

### Требования
- Node.js 18+
- npm / yarn

### Инструкция
1. **Клонирование и установка:**
   ```bash
   git clone [url-репозитория]
   cd admin-panel
   npm install
   ```
2. **Запуск разработки:**
   ```bash
   npm run dev
   ```
3. **Сборка для продакшна:**
   ```bash
   npm run build
   ```
