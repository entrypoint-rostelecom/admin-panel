# Admin Panel — Система управления доступом

Современная и производительная панель администратора для управления пользователями, контроля проходов и мониторинга системы безопасности. Проект разработан с акцентом на UX/UI, производительность и масштабируемость.

## 🌟 Основные возможности

- **Dashboard:** Наглядная статистика по активным пользователям, блокировкам и проходам за текущий день.
- **Управление пользователями:** Создание, удаление, а также "заморозка" (блокировка) учетных записей в реальном времени.
- **Мониторинг проходов:** Детальный лог всех событий с фильтрацией по сканерам, результатам и ФИО.
- **Журнал безопасности:** Полный аудит действий администраторов и системных событий.
- **Брендинг:** Оформление в корпоративном стиле.
- **Локализация:** Полная поддержка русского и английского языков (i18next).
- **Темная тема:** Поддержка светлого и темного режимов с сохранением предпочтений в LocalStorage.

## 🛠 Технологический стек

- **React 18** — интерфейс приложения.
- **Vite** — быстрая сборка и современный инструмент разработки.
- **TypeScript** — строгая типизация для повышения стабильности кода.
- **Redux Toolkit & RTK Query** — эффективное управление глобальным состоянием и кэширование запросов к API.
- **Styled Components** — гибкая и изолированная стилизация компонентов.
- **FSD (Feature-Sliced Design)** — современная архитектура, обеспечивающая чистоту и легкость поддержки кода.
- **i18next** — интернационализация приложения.

## 🚀 Быстрый старт

### Требования
- Node.js версии 18 или выше
- npm или yarn

### Установка и запуск
1. Установите зависимости:
   ```bash
   npm install
   ```
2. Запустите проект в режиме разработки:
   ```bash
   npm run dev
   ```
3. Для сборки проекта в продакшн:
   ```bash
   npm run build
   ```
